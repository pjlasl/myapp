About
"Terminal" was designed by a computer software engineer to assist him scanning for networks that could be used for nepharious reasons. The program scans nearby networks and provides the following: Network name, IP address, and the security protocol associated to the network. The original author has released as open source with the disclaimer that it should only be used for educational purposes only.
